Function atan2

Calculate the inverse tangent function with two arguments, y/x. By providing two arguments, the right quadrant of the computed angle can be determined.

For matrices, the function is evaluated element wise.

Syntax

atan2(y, x)

Parameters

Parameter Type Description
y number | Array | Matrix Second dimension
x number | Array | Matrix First dimension

Returns

Type Description
number | Array | Matrix Four-quadrant inverse tangent

Examples

atan2(2, 2) / pi;       // returns number 0.25

var angle = unit(60, 'deg'); // returns Unit 60 deg
var x = cos(angle);
var y = sin(angle);

atan(2);             // returns Complex 1.5707963267948966 -1.3169578969248166 i
